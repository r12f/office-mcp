# Office MCP Control Brand Mark

The Office MCP Control mark represents local office automation under user
control. The composition uses two abstract document or app panes as the work
surface, then runs a visible command spine through operator nodes into a control
dial. The dial reads as a live approval/control point instead of a settings
gear, so the mark can stand for controlling live Office documents without
looking like a generic preferences icon.

## Visual Metaphor

- Document panes: live Word and Excel files are present, but abstracted as
  neutral productivity surfaces rather than host-specific document silhouettes.
- Command spine: the blue route with two operator nodes makes control and local
  command routing visible even at tray size.
- Control dial: the yellow dial communicates user approval and command control
  without becoming a gear icon or settings metaphor.
- Dark shell: the dark rounded square gives the tray icon and installer icon a
  stable desktop utility silhouette at small sizes.

## Palette

- `#0F172A`: desktop utility shell and high-contrast stroke.
- `#EAF2FF`: neutral document pane surface.
- `#2457D6`: command spine and routing accent.
- `#48D6A4`: automation/control pane accent.
- `#F8D84A`: active control node.

The palette is intentionally not a Microsoft Office app palette. Host-specific
Word and Excel colors remain UI accents only and are not the product mark.

## Minimum Sizes

- 16 px: tray/menu-bar glyph must keep the dark shell, command spine, and
  yellow control dial legible enough to avoid reading as a blank square.
- 32 px: ribbon and Windows notification-area usage must show the document
  panes, command spine, and control dial.
- 80 px and above: Office command icon and installer usage should show all pane,
  routing, operator-node, and dial details.

## Non-Microsoft Distinction

The source artwork must not copy, trace, remix, or visually impersonate
Microsoft Office, Word, Excel, PowerPoint, Outlook, Windows, or Microsoft 365
marks. In particular, it must not use Office ribbon tiles, Microsoft 365
multi-color gradients, Word document silhouettes, Excel grid marks, Windows app
logo conventions, placeholder initials, or a standalone gear icon. The mark is
an original abstract control console for documents, with the command spine and
control dial as its signature elements.
